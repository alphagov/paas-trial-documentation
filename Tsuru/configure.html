Configuring your machine to use the Government PaaS

Configuring your machine is as easy as installing the correct CLI - start here: http://docs.tsuru.io/en/stable/using/install-client.html

Tsuru has 3 different apps that can be downloaded (crane, tsuru, tsuru-admin) you will only need the tsuru app.

For OSX you should download the darwin_amd64 binary.

Once you have downloaded and installed the CLI client for your OS you should be able to login to the environment using the following:

$ tsuru target-add trial https://trial-api.tsuru.paas.alphagov.co.uk -s

$ tsuru login

Email: your@email.gov.uk

Password: yourpassword

Now that you're setup head on over to the samples to try deploying an application.
